# FIle Mover
#####*Moves & copies files*

### Features
1. Moves files
2. Copies
3. List generate depending on your RegExp pattern

### Intended usage
1. Create files list by executing `createList.php`
2. Use `move.php` & feed that list file to it and choose how you want to copy or move your files

This steps will produce desired effect :)
